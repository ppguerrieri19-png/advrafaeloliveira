# 🏛️ Advocacia — Site Institucional

Site institucional estático de um escritório de advocacia, com design moderno, responsivo e animações suaves.

## 📁 Estrutura do Projeto

```
Advocacia/
├── index.html        # Página principal
├── style.css         # Estilos (responsivo, animações)
├── script.js         # Interações (menu mobile, reveal, FAQ)
├── assets/           # Imagens do projeto
│   ├── hero-office.jpg
│   └── lawyer-portrait.jpg
├── .gitignore
└── README.md
```

## ✨ Funcionalidades

- ✅ Design responsivo (mobile, tablet, desktop)
- ✅ Menu mobile com hambúrguer
- ✅ Animações on-scroll (fade-in / reveal)
- ✅ Seções: Hero, Sobre, Áreas de Atuação, Depoimentos, FAQ, Contato
- ✅ Formulário de contato
- ✅ Botão flutuante de WhatsApp

## 🚀 Como executar localmente

Basta abrir o `index.html` no navegador, **ou** usar um servidor local:

```bash
# Python 3
python -m http.server 8000

# Node.js (com npx)
npx serve .

# VS Code: extensão "Live Server"
```

Depois acesse: `http://localhost:8000`
